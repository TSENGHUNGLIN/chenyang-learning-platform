# 100室內設計網站分類架構研究

## 研究目的

根據晨陽設計老闆的要求，研究100室內設計網站（www.100.com.tw）的專業分類架構，並以此為基礎優化題庫的分類管理和標籤管理系統，確保題庫分類符合實際業界需求。

## 100室內設計網站的主要分類架構

根據網站研究，100室內設計網站的「裝修攻略」欄目包含以下8大主分類：

### 1. 裝前評估
**定義**：裝修前的規劃、預算、需求評估等準備工作

**應用場景**：
- 裝修預算規劃（預算怎麼抓？主要分為4大塊：室內設計與裝潢工程60-65%、家電設備15%、家具15%、軟裝家飾5-10%）
- 新成屋vs老屋翻新工期評估（新成屋2-4個月，老屋4-8個月）
- 預售屋交屋流程
- 裝修需求分析

### 2. 風格靈感
**定義**：各種室內設計風格的介紹、案例和靈感來源

**應用場景**：
- 風格分類：現代風、鄉村風、古典風、混搭風、北歐風、工業風、日式風、美式風、奢華風、新古典、新中式風
- 風格配色和材質選擇
- 風格元素和特點解析
- 粉色裝潢搭配技巧

### 3. 裝修知識
**定義**：裝修過程中的實用知識、技巧和注意事項

**應用場景**：
- 小宅輕隔間放大格局技巧（玻璃隔間、移動式拉門、書櫃收納櫃牆、屏風摺疊門）
- 壁紙vs油漆選擇（預算、耐用度、維護難易度、施工環境）
- 床頭掛畫風水禁忌與搭配法
- 開放式廚房油煙處理
- 浴室沒開窗濕氣處理
- 穿堂煞化解方法

### 4. 空間設計
**定義**：各種空間的設計規劃和布局技巧

**空間分類**：
- 客廳（家具擺放、招財布局）
- 餐廳
- 衛浴
- 書房
- 廚房（小廚房動線、收納）
- 臥室
- 玄關（二進式設計、收納）
- 陽台
- 吧檯
- 兒童房
- 衣帽間

**設計重點**：
- 動線規劃
- 收納設計
- 空間放大技巧
- 功能分區

### 5. 工程施工
**定義**：裝修施工過程中的工程技術和施工要點

**應用場景**：
- 系統傢俱施工6大隱藏雷區
- 視聽室裝潢（隔音、音響設計）
- 頂天收納櫃施工
- 半開放式衣櫃規劃
- 女兒牆（起源、法規、功能）

### 6. 建材百科
**定義**：各種建材的特性、選擇和應用知識

**應用場景**：
- 廚房系統櫃門板材質選擇（耐髒、好清、維修費）
- 不鏽鋼流理台選購（優缺點、選購指南、安裝保養）
- 茶色玻璃+玫瑰金搭配
- 地板材質
- 牆面材質
- 天花板材質

### 7. 家具家電
**定義**：家具和家電的選購、搭配和使用建議

**應用場景**：
- 人體工學沙發選購
- 家具擺放技巧
- 家電配置建議
- 智能家居整合

### 8. 軟裝搭配
**定義**：窗簾、抱枕、擺飾等軟裝元素的搭配技巧

**應用場景**：
- 色彩搭配
- 材質搭配
- 風格協調
- 季節性調整

## 優化建議

### 方案一：保持現有架構，擴充子分類

**優點**：
- 維持現有題庫結構，不需大規模調整
- 符合設計公司內部培訓需求（強調客戶關係、廠商管理、專業溝通等）

**建議新增子分類**：
1. 設計美學 > 新增「軟裝設計」
2. 工程管理 > 新增「產品知識」
3. 客戶關係 > 新增「專案規劃」
4. 專業溝通 > 新增「實務技巧」

### 方案二：整合100室內設計分類，建立混合架構

**優點**：
- 更貼近業界實務分類
- 便於從外部資源（如100室內設計網站）取材出題
- 考生更容易理解分類邏輯

**建議新架構**：

**A. 專業技能類（對應100室內設計）**
1. 裝前評估
2. 風格靈感
3. 空間設計
4. 工程施工
5. 建材百科
6. 家具家電
7. 軟裝搭配
8. 法規知識

**B. 職能素養類（晨陽設計特色）**
1. 客戶關係
2. 廠商管理
3. 專業溝通
4. 制度流程
5. 領導管理

### 方案三：建立雙層標籤系統

**優點**：
- 保持現有分類架構不變
- 透過標籤系統整合100室內設計分類
- 靈活性最高，可同時滿足內部培訓和業界對接需求

**實施方式**：
- **主分類**：保持現有8大分類（設計美學、工程管理、客戶關係等）
- **業界標籤**：新增8個標籤對應100室內設計分類（裝前評估、風格靈感、空間設計等）
- **其他標籤**：保持現有標籤體系（難度、職位、考核類型等）

## 實施建議

### 推薦方案：方案三（雙層標籤系統）

**理由**：
1. **最小化變動**：保持現有分類架構，不影響已建立的題庫
2. **最大化靈活性**：透過標籤系統整合業界分類，可同時滿足多種需求
3. **易於擴充**：未來可持續新增標籤，不需調整主分類架構
4. **便於AI分析**：AI可根據題目內容自動建議業界標籤

### 實施步驟

**第一階段：擴充標籤體系**
1. 新增「業界分類」標籤組（8個標籤）
2. 新增「空間類型」標籤組（10個標籤）
3. 新增「風格類型」標籤組（9個標籤）
4. 執行資料庫遷移

**第二階段：整合AI分析功能**
1. 修改AI出題功能，自動分析題目內容
2. 根據關鍵字自動建議「業界分類」標籤
3. 根據關鍵字自動建議「空間類型」標籤
4. 根據關鍵字自動建議「風格類型」標籤
5. 自動標記「AI生成」標籤
6. 記錄考題出處（檔案名稱、頁碼、段落）

**第三階段：優化使用者介面**
1. 在題庫管理頁面新增標籤篩選功能
2. 支援多標籤組合篩選
3. 顯示標籤統計資訊
4. 支援批次新增/移除標籤
